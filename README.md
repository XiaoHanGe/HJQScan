# HJQScan
基于LBXScan库二维码扫描 ，使用教程：http://blog.csdn.net/qq_31810357/article/details/50598644



## Usage

1.拖拽`JQScan`到你的项目中;

2.pod "ZXingObjC" ;

3.plist设置

- Privacy - Camera Usage Description            相机
- Privacy - Photo Library Usage Description     相册
## 注意

iOS9.0之后要在plist加两个权限:

- Privacy - Camera Usage Description            相机
- Privacy - Photo Library Usage Description     相册
